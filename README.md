# Sentra website alternative

An alternative Sentra landing page built in the visual language of the **Oberon**
reference template — warm paper (Pampas), burnt-orange accent, Archivo + Geist Mono
typography, dashed grid guides, index badges, corner brackets, and the signature
node-graph "Integrations" widget — with all copy tailored to Sentra
(public-discourse / narrative intelligence).

- `index.html` — the full single-file page (fonts via Google Fonts CDN).

Separate from the primary teal Sentra site (`sentra-site`).
